# GitHub Project Shortlist

This shortlist focuses on projects that could complement ElastiTune’s search-tuning and persona-review loops.

## 1. Elastic Search UI

- GitHub: https://github.com/elastic/search-ui
- Why it matters: ready-made search components, query state, and result rendering patterns.
- Possible integration: use it as a reference for advanced search UX or as a replacement for some of the custom result panels.

## 2. Searchkit

- GitHub: https://github.com/searchkit/searchkit
- Why it matters: opinionated React search UI plus Elasticsearch query helpers.
- Possible integration: compare its query composition model to ElastiTune’s profile generation flow.

## 3. LlamaIndex

- GitHub: https://github.com/run-llama/llama_index
- Why it matters: retrieval pipelines, document loaders, and evaluation utilities.
- Possible integration: borrow loader / chunking ideas for committee mode and future document ingestion.

## 4. Elasticsearch Labs

- GitHub: https://github.com/elastic/elasticsearch-labs
- Why it matters: contains Elastic-authored experiments and search demos, including product-store style examples.
- Possible integration: reuse benchmark ideas or index patterns when designing new demos.

## 5. TinyTroupe

- GitHub: https://github.com/microsoft/TinyTroupe
- Why it matters: agent/persona simulation patterns.
- Possible integration: use as inspiration for richer persona reactions or custom committee archetypes.
